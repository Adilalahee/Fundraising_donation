What is the document object model(DOM)?

The Document Object Model (DOM) is like a blueprint that represents the structure of a web page. It helps developers access and change different parts of the page, such as headings, paragraphs, images, and forms. Think of the DOM as a way to organize and control all the elements that make up a web page.

How do you select an element from the DOM?

One way of selecting an element is by using the .getElementById() method and passing the desired ID as an argument.

What is event delegation in the context of DOM and why it is useful?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element. It refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. This can be done on a particular target element using the .target property of an event object.

How do you manipulate an elements attributes and styles using the DOM?

Each DOM element has a element.style attribute, which is a JavaScript object. The element.style object has attributes that can be used to read or set style properties for the DOM element.
